# esp32-max30100-pulse-oximeter
**Heart rate and SpO₂ monitoring system using ESP32, MAX30100 and OLED SSD1306.**

Proyecto para medir la frecuencia cardíaca (BPM) y la saturación de oxígeno (SpO₂) usando un ESP32 y un sensor MAX30100.

**Características:**

Medición de BPM.

Medición de SpO₂.

Pantalla OLED SSD1306.

Comunicación I²C.

Filtro exponencial (EMA).

Animación de latidos.

**Hardware:**

ESP32 DevKit V1

MAX30100

OLED SSD1306

Protoboard

Cables Dupont
